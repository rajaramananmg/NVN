# Narthana Vinayagar Natyalaya — Website

A fully refined, production-ready single-page landing website.

## 🚀 Deploy in 3 Steps

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally to preview
```bash
npm run dev
```
Opens at http://localhost:3000

### 3. Build for production
```bash
npm run build
```
The `dist/` folder is ready to upload to any static host.

## ☁️ Free Hosting Options
- **Netlify**: Drag and drop the `dist/` folder at netlify.com/drop
- **Vercel**: Run `npx vercel` in this folder
- **GitHub Pages**: Push to GitHub, enable Pages from dist/
